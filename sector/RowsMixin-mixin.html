RowsMixin

RowsMixin<T> mixin

A mixin that provides a partial implementation of GridAxis for rows.

Given a reference to a Grid implementation, this mixin provides default implementations for most remaining methods in GridAxis that are not implemented, leaving only grid, removeAt, and insertAt up to the user.

Examples

class MyRows<T> extends GridAxis<T> with RowsMixin<T> {
  MyRows(this.grid);

  @override
  final Grid<T> grid;

  @override
  void removeAt(int index) { /* ... */ }

  @override
  void insertAt(int index, Iterable<T> row) { /* ... */ }
}
